# Selfcriticism LangGraph Project

## How to Run

### Step 1 - Clone the repo
```bash
git clone https://github.com/sjdasadi/selfcritisismlanggragh_project
cd selfcritisismlanggragh_project
```

### Step 2 - Create .env file
```bash
echo "OPENROUTER_API_KEY=your_api_key_here" > .env
```
> Replace `your_api_key_here` with your actual OpenRouter API key.

### Step 3 - Install dependencies
```bash
pip install langchain-openai python-dotenv langgraph
```

### Step 4 - Run the notebook
```bash
jupyter notebook
```
Then open `02-selfcritisistmlanggraph_improved.ipynb` in your browser.

## Get API Key
Go to [https://openrouter.ai](https://openrouter.ai) and create a free account to get your API key.
